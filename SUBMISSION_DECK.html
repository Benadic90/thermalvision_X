5 Core Innovations

1. Zero-Shot via FFT

2D Fast Fourier Transform separates structure (band-invariant) from color (band-specific). Train on visible, infer on IR.

2. Physics-Guided Loss

Planck's Law and Stefan-Boltzmann Law embedded as differentiable constraints. Colors are thermodynamically accurate.

3. Lightweight UNet (~5M params)

Runs at 15-30 FPS on edge devices. No cloud GPU needed. Deployable on UAVs over disaster zones.

4. Uncertainty Maps (MC Dropout)

Per-pixel confidence scores. Tells operators "95% sure here, 30% there." Critical for life-saving decisions.

5. ISRO-First Design

Native support for ISRO HDF5 formats (INSAT-3D, Oceansat-3, TRISHNA). Built for Indian satellite missions from Day 1.

Technical Architecture

Frequency Domain Decoupling (The Core Insight)

Images have two components in frequency space:

  • High Frequency = Edges, textures, structure --> SAME in visible AND infrared (band-invariant)
  • Low Frequency = Color, luminance --> EXISTS only in visible (band-specific)

We mask the low frequencies during training, forcing the UNet to reconstruct color from structure alone. At inference, IR images provide structure --> the model fills in physically accurate color.

Physics-Informed Loss Function

L_total = L_recon + L_focal_freq + L_planck + L_stefan + L_cosine

Where L_planck penalizes outputs that violate Planck's blackbody radiation law, and L_stefan enforces Stefan-Boltzmann total emissive power constraints.
